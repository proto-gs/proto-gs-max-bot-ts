# MAX Чат-бот проекта GS

Простой чат-бот для мессенджера MAX на базе `@maxhub/max-bot-api` и TypeScript.

Описание проектов gs и создание шифрования proto-gs

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Скомпилируйте TypeScript:
   ```bash
   npx tsc
   ```
3. Задайте переменную окружения `BOT_TOKEN` и запустите:
   ```bash
   node index.js
   ```
